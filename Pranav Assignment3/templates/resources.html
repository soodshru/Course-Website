Logging into Mathlab

  • For Mac, it already comes with the application "terminal". Simple click on it to run. Login to mathlab by typing ssh utorid@mathlab.utsc.utoronto.ca then simply enter your password.
  • For Windows you will need a SSH client. PuTTY is a good free one. You can download and install here:PuTTY Use the same command as above to login to mathlab: ssh utorid@mathlab.utsc.utoronto.ca
